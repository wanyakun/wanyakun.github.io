    CentOS7安装Maven

    wanyakun 6/24/2021

    Maven官方下载地址为：Maven下载 (opens new window)
    查看最新版本为3.8.1，

    # 下载

    下载安装包，选择tar.gz压缩包：

     wget https://mirrors.bfsu.edu.cn/apache/maven/maven-3/3.8.1/binaries/apache-maven-3.8.1-bin.tar.gz
    1

    # 解压

    ## 解压
tar vxf apache-maven-3.8.1-bin.tar.gz 

## 移动
sudo mv apache-maven-3.8.1 /usr/local/maven3
    1
    2
    3
    4
    5

    # 修改环境变量

     sudo vim /etc/profile
    1

    添加以下内容

    MAVEN_HOME=/usr/local/maven3
export MAVEN_HOME
export PATH=${PATH}:${MAVEN_HOME}/bin
    1
    2
    3

    执行source使环境生效

    source /etc/profile
    1

    # 验证

    mvn -v

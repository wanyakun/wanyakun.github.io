    frame、bound、center区别

    wanyakun 12/13/2022

    # 是什么

    # frame

    frame：该 view 在父 view 的坐标系统中的位置和大小。参照点是父 view 的坐标系统

    # bound

    bound：该 view 在自身坐标系统中的位置和大小。参照点是自身坐标系统。

    # center

    center：该 view 的中心点，在父 view 坐标系统中的位置和大小。参照点是父 view 点坐标系统。

    # 区别和影响

    iOS 是以 center 为基准的，一般通过改变 center 的位置改变（移动）视图的位置。另外对视图进行旋转，缩放也都是相对于 center 进行操作的。

    bound 像是浮于 frame 之上。frame 是一个框架，bound 是显示子 view 的东西。

    bound 的 point 不会改变 frame 的原点，改变的是自身的原点，进而影响子 view 的位置。这个作用更像是移动 bound 的原点。 对于 bound 的 size，它可以改变 frame。如果 bound 的 size 比 frame 的 size 大，frame 也会跟着变大，那么 frame 的原点也会改变，但 center 不会变。这个作用更像边界的意思。
